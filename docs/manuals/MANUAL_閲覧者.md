# マニュアル：閲覧者向け

このサイトで「できること」と基本的な使い方です。

## できること

| 項目 | 操作 | 場所 |
|------|------|------|
| イベント概要の確認 | トップのヒーローで日付・会場・料金を確認 | ページ上部 |
| タイムテーブルの確認 | 5.16 SAT と 5.17 SUN の両日が縦に並んで表示 | SCHEDULE セクション |
| タイムテーブルからアーティストへ移動 | 出演枠の名前をタップすると該当アーティストカードへスクロール＋ハイライト | SCHEDULE → ARTISTS |
| 出演アーティストの確認 | 写真・名前・会場・出演時間を一覧で閲覧 | ARTISTS セクション |
| アーティスト専用ページを見る | カードの写真または名前をタップ | 各アーティストカード |
| SNSでシェア | 専用ページ下部の X / LINE / Facebook / URLコピー ボタン | 各アーティスト専用ページ |
| 会場情報の確認 | 3会場それぞれの時間と特徴 | VENUES セクション |
| アクセス情報の確認 | 最寄駅と地図 | ACCESS セクション |
| ページ内移動 | ヘッダーのナビ（ABOUT / SCHEDULE / ARTISTS / VENUES / ACCESS） | 画面上部 |

## 操作手順

### 1. 出演者を時間帯から探す

1. ページ上部メニューの「SCHEDULE」をクリック
2. 表示したい日（5.16 SAT / 5.17 SUN）をタブで選択
3. 3つのステージそれぞれのタイムテーブルを確認

### 2. 出演者を写真で探す

1. ページ上部メニューの「ARTISTS」をクリック
2. グリッドから興味のあるアーティストを見つける
3. カード内のタグ（会場）と時間で出演日時を確認

### 3. 会場までのアクセスを確認する

1. 「ACCESS」セクションへ移動
2. 各会場の最寄駅を確認
3. 埋め込み地図で位置関係を把握

## できないこと

- チケット予約：本イベントは**入場無料**のため予約は不要です
- 出演者の動画・音源の再生
- コメント投稿・問い合わせフォーム送信
- 会員登録・ログイン

## スマートフォンでの閲覧

- モバイル表示に完全対応しています
- ハンバーガーメニュー（画面右上の3本線アイコン）からセクション間を移動できます

## 注意事項

- 出演者・時間・会場は**変更になる場合があります**
- 日比谷ステップ広場の特設ステージは**雨天開催・荒天中止**
